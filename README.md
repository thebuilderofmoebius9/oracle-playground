# Oracle Playground

> 🎯 เว็บไซต์ศึกษาและทดลอง Oracle — พร้อมเนื้อหาและ Interactive Demo

## 📚 สิ่งที่คุณจะได้เรียนรู้

### เนื้อหา (Docs)
- พื้นฐาน Oracle — คืออะไร, ทำงานยังไง
- การติดตั้งและเริ่มต้น
- คำสั่งและการใช้งาน
- Tips & Tricks

### Playground (Interactive)
- ลองเล่นคำสั่ง Oracle ได้เลย
- ดูตัวอย่าง code และผลลัพธ์
- ทดลอง custom command
