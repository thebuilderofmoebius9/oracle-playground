# คำสั่งพื้นฐาน

## 🎯 คำสั่งที่ใช้บ่อย

### `/who` — ใครอยู่?

แสดงข้อมูล Oracle ปัจจุบัน:

```bash
/who
```

ผลลัพธ์:
```
🤖 GB Oracle
📍 Location: gb-oracle
🎯 Role: Worker
👤 Human: Axe
```

### `/learn` — เรียนรู้

ศึกษา codebase หรือ project:

```bash
/learn <project-name>
```

### `/trace` — ตามรอย

ค้นหาและ discover ใน codebase:

```bash
/trace <query>
```

### `/rrr` — Session Retrospective

สรุปสิ่งที่ทำมา:

```bash
/rrr
```

## 💬 การสนทนา

### ถามคำถาม

พิมพ์ธรรมดา — Oracle จะตอบ:

```
อยาก setup database ยังไง?
```

### ขอให้ทำงาน

ใช้ imperatives:

```
สร้าง file ชื่อ test.txt
วิเคราะห์ logs/
สรุป PR นี้
```

## 📝 Memory

### จำสิ่งที่สำคัญ

Oracle จะจำสิ่งที่คุณบอก:

```
จำว่าฉันชอบใช้ vim ไม่ใช่ nano
```

### ดึงความทรงจำ

```
ฉันบอกว่าชอิน editor อะไร?
```
